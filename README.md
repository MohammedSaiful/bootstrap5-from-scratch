# Bootstrap 5

This repository contains projects created while completing the course "The Ultimate Bootstrap Guide – Bootstrap 5 from Scratch" by Fatah Gabrial on Udemy.
The course focuses on building modern, responsive websites using Bootstrap 5 and best practices in front-end development.

Throughout this course, I learned how to design responsive layouts using the Bootstrap grid system, implement various Bootstrap components, and build mobile-first user interfaces. The projects in this repository demonstrate practical usage of Bootstrap utilities, components, and responsive design techniques.

##  What I Learned
- Bootstrap 5 Grid System
- Responsive Web Design
- Bootstrap Components (Navbar, Cards, Buttons, Modals, etc.)
- Utility Classes for styling and layout
- Mobile-First Development Approach
- Building clean and modern UI layouts
